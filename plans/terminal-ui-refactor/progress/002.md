## Frame shape for the web consumer (task 004)

The `SnapshotFrame` pushed on `handleConnection` (before any live frame) has
this shape (from `@ddx/term-contract`, `packages/ddx-term-contract/src/ws-frames.ts`):

```ts
{
  type: 'snapshot';
  terminalId: string;
  data: string;          // captured pane content, RAW bytes (ESC/CR decoded, not \033/\015)
  cols: number;           // pinned 120
  rows: number;           // pinned 30
  withAnsi: true;
  scrollbackLines?: number; // bound used for the -S capture (default 500); undefined = viewport-only
}
```
