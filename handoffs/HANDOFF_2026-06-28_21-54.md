## Key Decisions
- **`clear()` is display-only** — tmux buffer is shared canonical state the MCP agent
  reads; a human clearing their view must NOT mutate agent-visible state.
- **`onExit` removes by closure-id, no windowId→terminalId lookup** — each XtermClient
  is bound to one terminalId (dual-attach model), so the closed terminal is unambiguous.
- **Kill = two-step inline confirm**, not `window.confirm` (design-system §6/§7 ban both
  native dialogs AND toasts). Modeled as a per-row 3-state machine.
- **Broker stays auth-free** — localhost-only by design (recorded in broker CLAUDE.md);
  the new PATCH follows that contract. ⚠ SECURITY REVIEW TARGET (see below).
- **Broker bundled via dist-copy, not tsup** — esbuild breaks NestJS decorators.
